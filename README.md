# 小宇在线工具箱

## 使用方法
1. 下载程序源码
2. 上传至服务器
3. 解压到网站根目录
4. 导入数据库文件
5. 修改数据库配置文件
6. 默认登录用户名和密码为：admin/123456


## 测试环境为
### 第一种
Nginx 1.21

PHP 7.2

MySQL 5.7

### 第二种
Nginx 1.14

PHP 5.6

MySQL 5.6


 
**压缩包内有各个路径说明，数据库每个字段都已经写好注释** 

**此版为最初版本，后续我会加入更多功能免费提供给大家使用**
